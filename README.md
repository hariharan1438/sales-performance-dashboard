# Sales Performance Dashboard

## Project Overview
This project analyzes retail sales performance using Power BI and the Superstore dataset.

## Tools Used
- Power BI
- Microsoft Excel

## Dashboard Features
- Total Sales KPI
- Total Profit KPI
- Sales by Region
- Profit by Region
- Sales Trend Over Time
- Sales by Category

## Key Insights
- West region generated the highest sales and profit.
- Technology category recorded the highest sales.
- Sales showed a positive growth trend from 2014 to 2017.
- South region generated the lowest sales performance.
